# Developer Guidelines

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js

## Coding Conventions
- **Functional Components:** Always use functional React components with hooks.
- **TypeScript:** Prefer TypeScript for all new code to ensure type safety.
- **Formatting & Linting:** Use ESLint and Prettier to maintain consistent code style.
- **Testing:** Write and run tests using Vitest.

## AI Assistant Guidelines
- **Explain Generated Code:** Provide clear explanations for any code generated or modified.

## Git & Version Control
- **Conventional Commits:** Follow the Conventional Commits specification for all commit messages.
